视觉参考 Prompt：JARVIS AI 工业未来主义界面规范
1. 核心设计理念 (Core Aesthetic)

风格定位： “极简工业未来主义” (Minimalist Industrial Futurism) 融合 “高端建筑蓝图” (High-end Architectural Blueprint)。

视觉感受： 精密、冷峻、高科技、但具有人文关怀的优雅感。

光影氛围： 在“实验室白”与“深空暗夜”之间切换，利用微弱的背景光晕 (Subtle Glow) 和磨砂玻璃 (Glassmorphism) 营造层次。

2. 色彩系统 (Color Palette)

主色调 (Primary):

Arch White (建筑灰白): #F2F2F2 (用于 Light Mode 背景，模拟高级纸张质感)

Deep Dark (深夜蓝灰): #1E1F2B (用于 Dark Mode 背景及 Light Mode 文字，呈现深邃的科技感)

强调色 (Accents):

Electric Purple (电光紫): #8A2BE2 (用于 AI/Agent 相关的激活状态、光晕)

Tech Cyan (科技青): #00AFB9 (Light Mode 下的科技点缀)

Neon Cyan (霓虹青): #00FFFF (Dark Mode 下的高亮、扫描线、数据流)

3. 排版体系 (Typography System)

混合字体策略 (Mixed Typography):

正文/通用 (Body): Inter —— 保持极高的可读性和现代感。

标题/数据 (Headings/Data): Space Grotesk —— 宽体无衬线字体，用于营造机械感、数据终端的视觉效果 (如 OCR Accuracy, System Status)。

艺术强调 (Editorial Accent): Playfair Display (Italic) —— 经典的衬线体斜体，仅用于 "Designed for Builders" 这类有人文温度的口号，形成强烈的“机械 vs 人文”对比。

排版特征： 巨大的字号对比 (8xl/10xl vs xs)，极细的字重 (ExtraLight) 搭配 紧凑的字间距 (Tracking Tighter)。

4. 关键 UI 组件视觉规范 (Component Visuals)

Hero Section (首屏):

背景： 极慢速流动的模糊光斑 (Blob)，叠加倾斜旋转的超大号水印文字 (Kinetic Typography)，透明度极低，仅作为纹理。

数据展示： 巨大的数字显示 (如 99.8%)，配合极细的进度条加载动画。

输入模拟： 打字机效果的终端输入框。

交互式列表 (Interactive Matrix List):

常态： 半透明，无左边框，箭头隐藏。

Hover/激活态： 不透明度 100%，左侧出现高亮色块 (紫/青)，箭头图标从左侧滑入，下方展开详情文字。

视觉反馈： 这里不要用复杂的 UI 动画，而是通过 Hover 列表项，右侧预览区域直接无缝切换对应的 产品演示视频 (Loop Video)。

视频预览容器 (Video Containers):

边框： 1px 的半透明边框，带有微弱的内阴影。

装饰： 顶部模拟 OS 窗口栏 (系统状态灯、文件名)，模拟一种“在操作系统内运行软件”的沉浸感。

遮罩： 视频上层叠加一层极淡的纹理或渐变，使其不突兀地融合进背景。

CTA 按钮 (Call to Action):

样式： 全圆角 (Full Rounded)，极简边框。

特效： Hover 时背景出现微弱的渐变光晕，文字变色，内部不仅有文字，还有一个脉冲式的小圆点 (Pulsing Dot) 表示“在线/活跃”。

5. 动效指导 (Motion Guidelines)

速度感：

背景氛围： 极慢 (120s duration)，几乎静止的流动，营造史诗感。

UI 交互： 快速干脆 (0.3s - 0.4s)，符合工具类软件的高效特性。

跑马灯 (Marquee): 匀速循环，无缝衔接。

扫描效果： 模仿雷达或扫描仪的垂直/旋转扫描线，用于强调“感知 (Sense)” 和 “数据捕捉”。

给工程师的一句话总结：

请还原一种类似“钢铁侠 JARVIS 系统”与“瑞士平面设计风格”结合的界面。重点在于字体组合的张力（Space Grotesk 的机械感 vs Playfair 的优雅感）、磨砂玻璃与细线条的精致度，以及视频预览与列表交互的丝滑配合。不需要复杂的 3D 渲染，用 CSS 的光影和排版来撑起高级感。
