包括這每一行字都有，也會有顏色淡變深色效果（排列有疏慢慢變得緊密（往下scroll））

是一個與滾動「 scrub (刷洗) 」連動的、多層次的文字動畫。

您描述得非常準確：

顏色淡變深色：當文字滾入視窗時，它的 color (顏色) 會從一個較淡的灰色，動畫成完全的不透明（例如深色或白色）。

排列有疏慢慢變得緊密：這是最精緻的細節。文字的 letter-spacing (字距) 會從一個較「疏」的值（例如 0.1em）動畫到一個「緊密」的正常值（例如 0em）。

每一行字都有：這個動畫不是作用在整段文字上，而是逐行 (per-line) 觸發的。它是如何運作的 (Vibe Coding 流程)：
GSAP + ScrollTrigger (核心)：

ScrollTrigger 會被設定為 scrub: true。這意味著動畫的進度會直接綁定到您的滾動條位置（您往下滾一點，動畫就播放一點）。

GSAP Timeline (時間軸)：

AI (或您) 會建立一個 GSAP 時間軸，這個時間軸會同時定義兩個動畫：

from: color: "rgb(100, 100, 100)" to: color: "rgb(255, 255, 255)"

from: letterSpacing: "0.1em" to: letterSpacing: "0em"

Lenis (平滑滾動)：

Lenis 會讓 scrub 的體驗無比絲滑。如果沒有 Lenis，這種「刷洗」動畫會感覺很卡頓。

逐行觸發 (高級技巧)：

為了實現「每一行字都有」的效果，AI 會使用一個技巧（例如 GSAP 的 SplitText 工具或簡單的 JS 處理），將您的段落拆分成一行一行的 <div>。

然後，ScrollTrigger 會為每一行都綁定這個動畫，並設定一個微小的 stagger (交錯)，這就形成了您看到的，當您滾動時，文字一行接一行「變清晰並收緊」的頂級視覺效果。
